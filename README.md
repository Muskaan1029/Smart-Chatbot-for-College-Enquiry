# College Enquiry Chatbot

## Description
This project is a **College Enquiry Chatbot** designed to help students get quick answers to common questions about courses, admission, fees, faculty, and other college-related information. It provides a **user-friendly interface** and automates responses, reducing the need for manual enquiry handling.

## Features
- Interactive chatbot for student queries
- Predefined answers for frequently asked questions
- Can be easily extended with new questions and responses
- Provides instant information about admissions, courses, events, and more

## Technologies Used
- HTML, CSS, JavaScript (for web-based chatbot)
- Optional: Python (for backend logic using Flask or chatbot framework)
- JSON (for storing questions and responses)

## Files
- `index.html` – User interface of the chatbot
- `style.css` – Styling for chatbot UI
- `app.js` – Logic for handling user input and generating responses
- `data.json` (optional) – Predefined questions and answers

## How to Run
1. Open the project folder.
2. Open `index.html` in a modern web browser (Chrome, Edge, Firefox).
3. Type your query in the chatbot input box to see responses.

## Future Enhancements
- Integrate AI/NLP for dynamic responses
- Add voice input and output
